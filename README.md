Step 1 : To clone a repository
Command -> git clone https://github.com/yashwanthk369/gitTest2.git
After this you can able to view the files on your repo in the VSCode.

Step 2 : If we create a file in VS and you want to add it to the github repo the command is 
Command -> git add .
this we copy all the files and copy it to your repo.

To check the status of your repo use command
Command -> git status

Step 3 : After the git add command we have to commit the changes to the repo by the command 
Command -> git commit -m "Inside the double quotes any type of message can be typed by its compulsory"

Step 4 : to final push all the files into your repo use the command 
Command -> git push origin main
Here, the main is the branch of your repo 

Step 5 : Now you can able to see the changes that you have done in the VS on your repo

Step 6 : vice versa if any changes made in the github as to be updated in the VS use the command
Command -> git pull
